게시물
  • [GET]

    • 게시물의 목록을 조회하는 API 입니다.

    권한 제한없이 누구나 접근할 수 있습니다.

  • [GET]

    특정 조건에 해당하는 게시물의 목록만을 조회/추출하는 API 입니다.

    권한 제한없이 누구나 접근할 수 있습니다.

    • boardNo:

      특정 게시판의 게시물 목록을 조회/추출할 수 있는 옵션

    • page:

      특정 게시판의 게시물 목록을 조회/추출할 수 있는 옵션

    • type:

      특정 게시판의 게시물 목록을 조회/추출할 수 있는 옵션

  • [GET]

    • 특정 게시물의 정보을 조회하는 API 입니다.

    권한 제한없이 누구나 접근할 수 있습니다.
    단, 웹 페이지의 링크를 통해 접근하거나, 특정 게시물의 고유 번호를 알고있어야 합니다.
    게시물 고유 번호는 0보다 큰 양수입니다.

  • [POST]

    • 새로운 게시물을 개설/생성하는 API 입니다.

    관리자 권한을 보유한 자만이 접근할 수 있습니다.

    • 기존 게시물을 수정하는 API 입니다.

    게시물 생성자 또는 관리자 권한을 보유한 자만이 접근할 수 있습니다.

    • 게시물을 삭제하는 API 입니다.

    게시물 생성자 또는 관리자 권한을 보유한 자만이 접근할 수 있습니다.
